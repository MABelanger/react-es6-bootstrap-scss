# react es6 bootstrap scss with webpack 2 sourcemaps
Minimum React + ES6 + Bootstrap + webpack 2 + sourcemaps that display H1 `Hello World!`

Install dependencies
```
$ npm i
```

To run webpack dev server:
```
$ npm run dev
```
